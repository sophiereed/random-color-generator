Random Color Generator

Description:

This may not look like much, but this is a GUI to randomize the background color. Try refreshing the page to see it in action.

Parameters:

These are the custom controls I added to customize my random color function. You can change the sliders on the right to change the GUI!

  • r: this changes the red tone variables.
  • g: this changes the green tone variables.
  • b: this changes the blue tone variables.
